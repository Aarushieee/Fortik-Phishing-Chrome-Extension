This project is a Chrome browser extension designed to help users identify and avoid phishing websites. It analyzes URLs and basic webpage indicators to flag potentially malicious or suspicious sites, promoting safer browsing practices.
The extension focuses on user awareness and cybersecurity fundamentals, providing alerts when a website exhibits common phishing characteristics. Built as a learning-focused security project, it demonstrates practical application of web technologies and phishing detection concepts.

Key Highlights
  Detects potentially suspicious or phishing URLs
  Provides real-time alerts to users while browsing
  Lightweight and easy-to-use Chrome extension
